# Changelog

All notable changes to AgentOps Mesh will be documented in this file.

## [v2.8] — 2026-06-20

### Added
- Public launch candidate
- GitHub Pages finalization
- Architecture Decision Records (ADRs)
- 6 governance dimensions: policy enforcement, approval gates, audit trail, cost monitoring, launch readiness, evaluation
- Hash-chained tamper-proof audit trail
- Sandboxed connectors (disabled by default)

### Changed
- Repository cleanup for public launch
- Documentation restructured for external contributors

## [v2.0] — 2026-05

### Added
- Deterministic policy engine (Python code, not prompts)
- Connector-agnostic architecture
- Launch readiness as code

## [v1.0] — 2026-04

### Added
- Initial governance framework
- Policy, audit, and evaluation modules
