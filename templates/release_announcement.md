# Release Announcement: Open Enterprise AgentOps Mesh v1.0

Open Enterprise AgentOps Mesh v1.0 is now available as a public stable MVP.

It provides an open-source AgentOps control-plane foundation for governing, evaluating, operating, and auditing enterprise AI agents.

## Highlights

- Evaluation Lab
- Governance Workflow Engine
- Agent Registry and Evidence Vault
- Policy-as-Code Guardrails
- Runtime Enforcement Hooks
- Observability Trace Ledger
- Connector/Tool Sandbox
- Procurement Control-Plane Demo
- GitHub Pages public site
- FastAPI backend prototype

## Positioning

This is not another chatbot. It is a control plane for deciding whether an enterprise AI agent should be allowed to move from idea to pilot to production.
