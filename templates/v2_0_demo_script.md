# v2.0 Demo Script

## Opening

This project is an open-source AgentOps control plane for governing, evaluating, operating, and auditing enterprise AI agents before production.

## Demo sequence

1. Open `site/control_plane_console.html`.
2. Show the capability map.
3. Open FastAPI `/docs`.
4. Call `/control-plane/release-status`.
5. Call `/control-plane/demo-flow`.
6. Run the procurement accelerator scenario.
7. Show provider gateway and model safety pages.
8. Show audit and observability summaries.
9. Explain why live execution remains disabled.

## Closing message

The framework is not trying to be another chatbot. It is the governance and runtime-control layer that enterprises need before letting agents interact with tools, data, models, and workflows.
