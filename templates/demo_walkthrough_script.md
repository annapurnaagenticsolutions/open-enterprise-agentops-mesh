# Demo Walkthrough Script

## Audience

- Executive
- Enterprise architect
- Governance leader
- Contributor

## Opening

This project is an open-source AgentOps control plane. It governs, evaluates, traces, and audits enterprise AI agents before production.

## Walkthrough path

1. Start with the business problem.
2. Show governance classification.
3. Show policy-as-code decision.
4. Show approval and evidence.
5. Show sandbox execution.
6. Show audit trail.
7. Show readiness report.

## Closing

The value is not that an agent can act. The value is that the enterprise can decide when, why, how, and under what evidence an agent is allowed to act.
