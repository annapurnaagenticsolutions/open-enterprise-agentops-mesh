# v2.3 Forward Roadmap

## Recommended next release

**v2.3 — Scenario Library and Benchmark Harness**

## Why

After v2.1, the project is easier to understand and contribute to. The next value increase should be reusable scenario depth, not live execution.

## Proposed scope

- scenario catalog format
- governance benchmark scenarios
- policy benchmark scenarios
- provider-routing benchmark scenarios
- model-safety benchmark scenarios
- procurement accelerator benchmark cases
- CLI benchmark runner
- deterministic benchmark score report
- static benchmark dashboard

## Still excluded

- live connector execution
- live provider execution
- production auth
- real secrets
- enterprise system writes

## Later releases

| Release | Candidate focus |
|---|---|
| v2.3 | Scenario Library and Benchmark Harness |
| v2.3 | Postgres/SQLite persistence adapter prototype |
