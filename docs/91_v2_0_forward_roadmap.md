# v2.0 Forward Roadmap

## Recommended direction

v2.0 should be a public architecture stabilization release, not a feature explosion.

Recommended v2.0 theme:

> Production-readiness architecture consolidation for the AgentOps Control Plane.

## Candidate v2.0 capabilities

1. Unified control-plane dashboard navigation
2. End-to-end demo flow linking procurement, provider routing, safety review, policy, approval, audit, and observability
3. SQLite optional storage adapter
4. API route grouping and OpenAPI documentation polish
5. CLI demo runner
6. Repository cleanup and package metadata normalization
7. Public website rewrite with stronger industry narrative

## Capabilities to delay

Delay these until v2.x after v2.0:

- live OpenAI/Anthropic/Gemini provider execution
- live ERP/ticket/email connector execution
- production secrets-manager integration
- enterprise SSO/OIDC integration
- real-time event streaming
- hosted SaaS deployment

## Reason

The project now has enough breadth. v2.0 should improve coherence, navigation, installation confidence, and public credibility.
