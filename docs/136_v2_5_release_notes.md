# v2.5 Release Notes

## Release name

**Public Feedback Loop and Community Intake**

## Added

- Community intake manifest
- Feedback channel catalog
- Use-case submission workflow
- Architecture critique workflow
- Adoption feedback model
- Roadmap feedback model
- Triage policy
- Community intake console
- Community readiness APIs
- Community readiness report script
- Tests and smoke-test coverage

## Boundaries

v2.5 does not add:

- live connector execution,
- live model-provider calls,
- real GitHub synchronization,
- CRM/community-platform integration,
- external identity integration,
- raw secrets.

## Decision

v2.5 is an adoption-readiness release. It improves the project’s ability to gather structured community signal before adding more runtime capability.
