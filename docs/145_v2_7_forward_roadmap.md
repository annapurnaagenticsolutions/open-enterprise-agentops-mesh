# v2.7 Forward Roadmap

## Recommended next release

v2.7 should be **Release Evidence Pack and Demo Recording Readiness**.

## Why

After v2.6, the project is easier to understand publicly. The next adoption step is to package evidence:

- demo walkthrough checklist,
- screenshot checklist,
- release evidence index,
- public demo script variants,
- benchmark evidence summary,
- executive one-page brief,
- maintainer launch checklist.

## What v2.7 should not do

Do not add live provider calls or live connector execution yet. The public narrative and release evidence should be stabilized first.
