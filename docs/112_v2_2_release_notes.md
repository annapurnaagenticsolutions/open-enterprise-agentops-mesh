# v2.3 Release Notes

## Release title

Scenario Library and Benchmark Harness

## Summary

v2.3 adds reusable benchmark scenarios and deterministic benchmark execution to improve project credibility, contribution quality, and release confidence.

## Added

- Scenario library
- Benchmark suites
- Benchmark weights
- Benchmark service
- Benchmark run persistence
- Benchmark API endpoints
- Static benchmark console
- CLI benchmark runner
- Scenario contribution guide
- Benchmark scoring model
- Release scorecard

## Still disabled

- Live model providers
- Live connectors
- Real enterprise credentials
- Raw secret storage
- Production execution

## Release decision

v2.3 is suitable as an adoption-quality release. It makes the project easier to test, extend, and discuss publicly.
