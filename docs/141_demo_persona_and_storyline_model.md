# Demo Persona and Storyline Model

## Personas

v2.6 defines four primary personas:

- Enterprise AI Leader
- Enterprise Architect
- AI Governance Owner
- Open-source Contributor

Each persona has a different success criterion. The site should not force them into the same path.

## Storyline

The primary storyline is:

> Before an enterprise agent touches production systems, the control plane evaluates its suitability, checks policies, validates approvals, executes only in sandbox, and records audit evidence.

## Why procurement remains the first demo

Procurement is business-readable. PO, invoice, challan, vendor, and exception-review flows make governance visible without requiring deep AI background.
