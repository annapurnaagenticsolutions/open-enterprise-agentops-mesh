# Demo Recording Script

## Target duration

Five to seven minutes.

## Audience

- Enterprise AI leaders
- Solution architects
- Data and platform leaders
- Governance and risk stakeholders
- Open-source contributors

## Opening narrative

Enterprise AI teams are moving from assistants to task-specific agents. The hard part is not creating a demo agent. The hard part is governing whether an agent should exist, what it can access, what model it can use, what evidence is required, and what happens when risk is detected.

Open Enterprise AgentOps Mesh demonstrates an open-source control-plane approach for this problem.

## Recording flow

1. Show the public landing page.
2. Explain the control-plane problem in one slide or paragraph.
3. Open the interactive demo path.
4. Walk through the procurement governed-agent scenario.
5. Show governance, policy, approval, model-routing, safety, and audit controls.
6. Show benchmark and validation evidence.
7. Show the release evidence console.
8. Close with what is intentionally disabled: live connectors, live model calls, raw secrets, and production IAM.

## Closing statement

This project is intentionally control-plane-first. It is designed to help enterprises assess, govern, evaluate, route, and audit agents before production execution.
