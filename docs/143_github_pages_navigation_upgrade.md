# GitHub Pages Navigation Upgrade

## Recommended GitHub Pages entry points

- `site/index.html` — main landing page
- `site/interactive_demo_path.html` — default public demo
- `site/control_plane_console.html` — architecture-oriented overview
- `site/api_catalog.html` — API surface
- `site/community_intake_console.html` — contribution and feedback

## Manual launch review

Before public sharing:

1. Open the site locally.
2. Start from the homepage.
3. Complete the procurement guided demo path.
4. Verify each linked console loads.
5. Confirm the live-execution boundary appears.
6. Confirm README links match the site paths.
