# v2.4 Five-Minute Demo Script

## 0:00–0:30 — Positioning

Open Enterprise AgentOps Mesh is an open-source AgentOps control plane. It helps enterprises decide whether AI agents are governed, evaluated, auditable, and ready before production.

## 0:30–1:15 — The problem

Show `site/index.html`.

Explain that most organizations can build agents, but struggle with risk, data readiness, tool access, approvals, cost, audit, and repeatable evaluation.

## 1:15–2:00 — Capability map

Show `site/control_plane_console.html`.

Explain the control-plane capabilities from intake to audit.

## 2:00–3:00 — Governance and policy

Show `site/governance_workflow.html` and `site/policy_workbench.html`.

Explain that risky actions should be allowed, controlled, approved, or denied deterministically.

## 3:00–4:00 — Business accelerator

Show `site/procurement_accelerator.html`.

Explain PO, invoice, challan, vendor, quantity, goods receipt, policy, sandbox, and readiness report.

## 4:00–4:40 — Audit and benchmarks

Show `site/audit_console.html` and `site/benchmark_console.html`.

Explain traceability and repeatable control-plane benchmarks.

## 4:40–5:00 — Closing

End with `site/public_launch_console.html`.

Message: this is a reference control plane for serious enterprise agent programs, not a chatbot demo.
