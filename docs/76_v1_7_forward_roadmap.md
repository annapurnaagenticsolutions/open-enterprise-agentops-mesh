# v1.7 Forward Roadmap: Live Connector Governance Pack

v1.7 should prepare the governance path for real connectors without enabling live execution.

Recommended scope:

- live connector readiness checklist
- rollback/compensation contract template
- environment promotion gates
- connector risk scorecard
- production evidence checklist
- external secret manager integration plan
- immutable audit retention plan
- connector kill-switch pattern
- incident response template

The platform should not enable live connector calls until these controls are modeled and tested.
