# Compliance Evidence & Audit Readiness

Problem
- Audits require extensive evidence; manual collection is error-prone and time-consuming.

Agentic Solution (Pattern)
- Compliance & Governance Agent: automate evidence collection, policy checks, and generate audit-ready reports; supports policy-as-code.
- Data & Integrations: policy engines, logs, CI/CD, ticketing, security tooling.
- HITL gating for sensitive evidence handling; compliance dashboards for stakeholders.

Success Metrics
- Time to evidence readiness, audit pass rate, policy drift rate.

Risks and Mitigations
- Handling PII; implement masking/pseudonymization and strict access controls.

Example Prompts/Templates
- Evidence collection prompts, policy validation prompts, report packaging prompts.
