# Cloud Cost Optimization at Scale

Problem
- Unoptimized cloud spend due to underutilized resources and governance gaps.

Agentic Solution (Pattern)
- Cloud Cost & Operations Agent: anomaly detection, right-sizing recommendations, and automated cost-control workflows.
- Data & Integrations: cloud billing data, CMDB, approvals/workflow tools.
- HITL gating for production changes; safe automation windows.

Success Metrics
- Cost savings realized, anomaly precision, automation adoption rate.

Risks and Mitigations
- Incorrect actions affecting performance; implement time-bound automation and rollback.

Example Prompts/Templates
- Cost anomaly prompts, right-sizing action prompts.
