# Learning & Development Automation

Problem
- Skill gaps mapping, personalized learning paths, and content curation are manual and slow to adapt to evolving needs.

Agentic Solution (Pattern)
- Learning & Development Agent: analyzes skills inventory, maps to competencies, recommends learning paths, curates content, and tracks progress.
- Components: talent/competency profiling, content recommendations, LMS integration, progress tracking, and knowledge seed.
- HITL gating for content curation when content is new or high-stakes.

Data & Integrations
- HRIS, LMS, content catalogs, performance data, feedback systems.

Interaction pattern
- Periodic skill assessments trigger learning path suggestions; agent curates and assigns content; progress updated in LMS.

Success Metrics
- Skill coverage, time-to-competency, completion rates, training ROI.

Risks and Mitigations
- Content quality; implement human review for new courses; track usage metrics to avoid low-value content.

- Example Prompts/Templates
- Learning path recommendation prompts, content curation prompts, progress reporting prompts.

## Experimental validation plan
- Offline evaluation: analyze historical learning paths and outcomes to calibrate recommendations and content curation.
- Pilot: 2-3 teams; 4-6 weeks; measure time-to-competency, completion rates, and ROI of learning programs; assess HITL workload.
- Scale: expand to more roles/domains; monitor content adoption and learning outcomes across tenants.

- Metrics
- Time-to-competency, completion rate, learning ROI, content adoption rate, HITL workload.
- Acceptance criteria: time-to-competency improvement, high-quality content adoption, HITL within capacity.

- Next steps
- Run pilot, refine prompts, align with LMS data, and scale.
- Learning path recommendation prompts, content curation prompts, progress reporting prompts.
