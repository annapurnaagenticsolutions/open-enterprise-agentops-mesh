# v2.6 Release Notes

## Release name

Public Site UX Polish and Interactive Demo Path

## Summary

v2.6 improves the adoption layer. It introduces a clearer public-site structure, audience-based navigation, interactive demo paths, persona definitions, UX copy guidance, public-site APIs, and release evidence.

## Added

- Public site navigation artifacts.
- Guided demo paths.
- Demo personas.
- UX copy and empty-state model.
- Public-site UX console.
- Interactive demo path page.
- Public-site UX API service.
- Tests and smoke checks.

## Not added

- Live model-provider calls.
- Live connector execution.
- Raw secrets.
- Production IAM.
- Production deployment.
