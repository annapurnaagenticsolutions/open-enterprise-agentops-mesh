# Public Launch Evidence Checklist

## Before publishing

- README explains the control-plane narrative.
- Static site has clear entry points.
- Backend tests pass.
- API smoke checks pass.
- Benchmark harness passes.
- Public repo validation passes.
- Evidence console is available.
- Demo recording checklist is complete.
- Live connectors and live providers remain disabled.
- No raw secrets are present.

## Launch proof to capture

- Screenshot of landing page.
- Screenshot of interactive demo path.
- Screenshot of release evidence console.
- Test result summary.
- Smoke check summary.
- Benchmark summary.
- Public repo validation summary.

## Recommended public wording

Open Enterprise AgentOps Mesh is an open-source AgentOps control plane for governing, evaluating, operating, and auditing enterprise AI agents before production.
