# v0.2 Release Notes

## Release theme

v0.2 converts the project from a static framework package into the beginning of a real open-source AgentOps framework.

The key addition is the **Agent Evaluation Lab**, supported by schemas, sample scenarios, deterministic scoring logic, and a backend/frontend scaffold.

## Added

- Evaluation Lab design document
- Framework evolution strategy
- Product architecture document
- Evaluation schema
- Sample agent scenarios
- Failure-mode library
- Certification levels
- FastAPI backend skeleton
- Pydantic request/response schemas
- Deterministic evaluator service
- Risk classifier service
- Certification service
- Frontend prototype
- Static evaluation lab HTML page
- Business accelerator example folders

## Excluded

- Self-Healing DevOps Intelligence Platform remains excluded from this track.
- Live LLM execution is not included in v0.2.
- Enterprise connectors are not included in v0.2.
- Persistent database storage is not included in v0.2.
