# Executive Demo Storyboard

## 1. Start with the problem
Pipeline failures waste time because context is scattered across TeamCity, GitHub, Jira, chat, and tribal knowledge.

## 2. Show dashboard/index.html
Show that the platform handles multiple failure classes, not only dependency issues.

## 3. Show dashboard/realtime.html
Show the near-real-time operational loop.

## 4. Show dashboard/impact.html
Show LLM decision rationale, evidence, alternatives, and long-term benefit.

## 5. Show dashboard/showcase.html
Explain architecture strength and why this is not a one-off bot.

## 6. Show dashboard/governance.html
Show that unsafe automation is blocked.

## 7. Show dashboard/value-simulator.html
Translate impact into triage hours, cost, and recurrence reduction.

## 8. Show dashboard/future-integrations.html
Explain that TeamCity + GitHub + Jira is the first adapter set, not the limit.

## Closing Message
This platform diagnoses failures, proposes safe action, blocks unsafe automation, learns from validated outcomes, and improves the delivery system over time.
