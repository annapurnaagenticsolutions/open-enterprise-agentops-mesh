# Release Notes — v0.5.1

## Release type

Client-facing polish and presentation correction.

## Why this release was needed

v0.5 improved the structure but still exposed internal build language on public pages. That could create the wrong impression during client showcasing because it described how the site was packaged rather than the business value it delivers.

## Key changes

- Reframed public site as **Agentic Solutions Studio**.
- Removed visible page-number labels from navigation and section kickers.
- Removed public version/release/corrective-build wording.
- Replaced “Pitch” with **Overview**.
- Replaced “Simulators” with **Solution Previews** in navigation.
- Replaced internal phrases like “built to prove” with client-facing value language.
- Changed visible output label from “Isolation rule” to **Scope boundary**.
- Changed visible output label from “Pitch note” to **Client value note**.
- Preserved all five existing solution previews.
- Retained Business scenario dynamic field updates.

## No feature expansion

No new solution preview was added. The focus was professional presentation and client-readiness.
