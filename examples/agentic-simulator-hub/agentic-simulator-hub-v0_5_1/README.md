- `simulators.html` — Solution Previews

## v0.5.1 focus

This release removes internal build language from the public pages and reframes the site for client pitching. Visible labels now read as a polished solution portfolio rather than a development notebook. Technical versioning, validation notes, and corrective details remain inside documentation only.

## Run locally

Open `index.html` in a browser. No backend, build step, or external dependency is required.

## Deployment

Upload the folder contents to GitHub Pages or any static hosting service.
