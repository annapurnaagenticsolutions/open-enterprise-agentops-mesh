# AgentOps Mesh Frontend Prototype

This is a plain HTML/CSS/JS frontend prototype for v0.2.

It is intentionally build-tool-free so that it can be opened directly in a browser and later converted to React + Vite when the interaction model stabilizes.

## Purpose

- Load sample agent scenarios
- Display weighted evaluation score
- Show certification level
- Show risk and governance blockers
- Explain the planned frontend direction

## Future frontend direction

- React + Vite workspace
- Use Case Canvas
- Evaluation Lab
- Governance Board
- Data Readiness dashboard
- Accelerator gallery
- Report export
